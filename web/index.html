Command Line Interface

Direct access to the entropy generation pipeline

# Generate passwords with NIST validation (fast mode)
python3 tests/scripts/RadioNoise.py -l 20 -n 5

# Generate passphrases (6 words)
python3 tests/scripts/RadioNoise.py -p -l 6 -n 3

# Full NIST test suite (15 tests, slower)
python3 tests/scripts/RadioNoise.py --full-test -n 5

# Test entropy quality only
python3 tests/scripts/RadioNoise.py -f entropy.bin --test-only

# Use RDSEED (direct CPU entropy) instead of RDRAND
python3 tests/scripts/RadioNoise.py --use-rdseed -n 5

Based on NIST SP 800-22 standards for random number generators
